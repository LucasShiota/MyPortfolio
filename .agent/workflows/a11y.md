This workflow helps ensure your portfolio is accessible to everyone.

### 1. Preparation

1. Ensure the development server is running: `npm run dev`.
2. The site should be accessible at `http://localhost:4321`.

### 2. Execution

// turbo

1. Run `npm run a11y`.
2. Review the output for any errors (Level AA compliance by default).

### 3. Fixing

1. If errors are found, investigate the specific element.
2. Common fixes: Add `aria-label`, improve color contrast, or add `alt` tags to images.
3. Re-run the scan until it passes.
